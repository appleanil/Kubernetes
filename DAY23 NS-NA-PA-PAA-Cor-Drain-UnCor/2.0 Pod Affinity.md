🔥 First Label Nodes

🔥 Common YAML Structure
affinity:
  nodeAffinity:
    requiredDuringSchedulingIgnoredDuringExecution:
1️⃣ In
Meaning:

Given list lo unna value match avvali.

Example:

env=prod or env=stage

apiVersion: v1
kind: Pod
metadata:
  name: pod-in
spec:
  affinity:
    nodeAffinity:
      requiredDuringSchedulingIgnoredDuringExecution:
        nodeSelectorTerms:
        - matchExpressions:
          - key: env
            operator: In
            values:
            - prod
            - stage
  containers:
  - name: nginx
    image: nginx
Telugu:

👉 env prod లేదా stage ఉన్న node లో deploy అవుతుంది.

👉 cpu label 4 కన్నా ఎక్కువ ఉండాలి.
